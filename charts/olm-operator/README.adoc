= Operator Lifecycle Manager operator template chart

*Current Chart Version*: `0.2.0`

== Usage

.`Charts.yaml`
[source,yaml]
-----
dependencies:
  - name: olm-operator
    repository: https://charts.hknlab.io
    version: 0.2.0
    alias: operator
    condition: operator.enable
-----

.`values.yaml`
[source,yaml]
-----
operator:
  enabled: true

  name: my-operator

  namespace:
    enabled: true
    name: my-operator-namespace
    annotations: {}
    labels: {}

  install:
    installPlanApproval: Automatic

  catalog:
    channel: stable
    source: my-operator-index
    sourceNamespace: olm

  operatorGroup:
    enabled: true
    namespace: my-operator-namespace
    additionalNamespaces: # []
      - my-managed-namespace

  config: {}

-----

Use the empty list (`[]`) to denote that the operator watches all namespaces. Otherwise, list namespaces that the operator watches.
